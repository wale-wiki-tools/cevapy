# CevaPy
A Wikipedia helper script to update [football season pages](https://en.wikipedia.org/wiki/Category:Seasons_in_association_football).

I built this script as a way to help automate the grind of updating match tables on each game day, and make it easier for new Wikipedia editors to contribute.

## Roadmap
- [x] Basic [sports table](https://en.wikipedia.org/wiki/Module:Sports_table/WDL) support.
- [ ] Basic [sports results table](https://en.wikipedia.org/wiki/Module:Sports_results) support
- [ ] Season infobox updating support
- [ ] Auto-uploading to Wikipedia
  - [ ] "Dry run" (test) support
- [ ] Repository & TOML schema for league configurations

## The name
CevaPy is named after the Balkan staple of [ćevapi](https://en.wikipedia.org/wiki/%C4%86evapi). Due to the historical immigration of families of former Yugoslavian countries to Australia, and many of them starting their own football clubs - foods like ćevapi have become a staple within the football community of Australia, both inside of these clubs and outside.
